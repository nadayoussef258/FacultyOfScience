# صفحة تواصل معنا - Contact Us Page

## نظرة عامة
تم تطوير صفحة تواصل معنا بتصميم حديث ومتجاوب باستخدام Angular و PrimeNG مع دعم كامل للغة العربية (RTL).

## المكونات (Components)

### 1. ContactPage Component
- المكون الرئيسي الذي يحتوي على جميع المكونات الفرعية
- يدعم التصميم المتجاوب والتنقل السلس

### 2. TopSection Component
- قسم البطل (Hero Section) مع تصميم جذاب
- يحتوي على:
  - عنوان "تواصل معنا"
  - وصف ترحيبي
  - مسار التنقل (Breadcrumb)
  - تأثيرات بصرية متحركة

### 3. ContactForm Component
- نموذج تواصل تفاعلي مع التحقق من صحة البيانات
- الحقول المتاحة:
  - الاسم الكامل (مطلوب، أكثر من حرفين)
  - البريد الإلكتروني (مطلوب، تنسيق صحيح)
  - رقم الهاتف (مطلوب، أرقام فقط)
  - الرسالة (مطلوبة، أكثر من 10 أحرف)
- معلومات الاتصال:
  - أرقام الهاتف
  - العنوان
  - البريد الإلكتروني
  - روابط وسائل التواصل الاجتماعي

### 4. Map Component
- خريطة تفاعلية لموقع كلية العلوم - جامعة أسوان
- معلومات إضافية:
  - العنوان التفصيلي
  - ساعات العمل
  - معلومات المواصلات

## المميزات التقنية

### التصميم
- دعم كامل للغة العربية (RTL)
- تصميم متجاوب لجميع الأجهزة
- استخدام PrimeNG للمكونات التفاعلية
- تدرجات لونية حديثة وتأثيرات بصرية

### التفاعل
- التحقق من صحة البيانات في الوقت الفعلي
- رسائل خطأ واضحة ومفيدة
- تأثيرات الحركة والانتقال
- تجربة مستخدم سلسة

### الأداء
- مكونات مستقلة (Standalone Components)
- تحميل كسول للخريطة
- تحسين الصور والموارد

## الملفات المحدثة

### HTML Templates
- `contactPage.component.html` - الصفحة الرئيسية
- `topSection.component.html` - قسم البطل
- `contactForm.component.html` - نموذج التواصل
- `map.component.html` - قسم الخريطة

### CSS Styles
- `contactPage.component.css` - أنماط الصفحة الرئيسية
- `topSection.component.css` - أنماط قسم البطل
- `contactForm.component.css` - أنماط النموذج
- `map.component.css` - أنماط الخريطة

### TypeScript Components
- جميع المكونات محدثة لتكون standalone
- إضافة imports للمكتبات المطلوبة
- تحسين منطق التحقق من البيانات

## التحسينات المضافة

1. **تصميم حديث**: استخدام تدرجات لونية وتأثيرات بصرية
2. **أيقونات تفاعلية**: إضافة أيقونات PrimeNG للحقول
3. **رسائل خطأ محسنة**: تصميم أفضل لرسائل التحقق
4. **خريطة تفاعلية**: خريطة Google Maps لموقع الجامعة
5. **معلومات شاملة**: تفاصيل الاتصال والموقع
6. **تجربة مستخدم محسنة**: انتقالات سلسة وتفاعل أفضل

## كيفية الاستخدام

1. تأكد من تثبيت PrimeNG في المشروع
2. استورد المكونات في الصفحة المطلوبة
3. استخدم `<app-contactPage></app-contactPage>` لعرض الصفحة كاملة

- دعم RTL في المتصفح
